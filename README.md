# nyc-taxi-numpy-analysis
NYC Taxi Data Analysis using NumPy

Overview

This project analyzes real-world NYC taxi ride data using NumPy to perform efficient numerical computations and extract meaningful insights.

The focus is on leveraging vectorized operations to handle large datasets without relying on traditional loops, improving both performance and scalability.


Dataset

Dataset: NYC Taxi Trips
Contains information such as:
Trip distance
Trip duration
Fare amount
Pickup month
Drop-off location


Key Objectives

1. Calculate average speed of taxi rides
2. Analyze ride distribution across months
3. Identify high-value trips (>$50)
4. Examine airport drop-off trends (JFK)


Technologies Used

Python
NumPy


Key Insights

1. Computed mean speed using distance and trip duration
2. Extracted total rides in February to observe monthly patterns
3. Identified number of rides with fare greater than $50
4. Analyzed frequency of rides ending at JFK Airport

2. Improved skills in data filtering and slicing
3. Learned efficient ways to process large datasets
4. Strengthened understanding of real-world data analysis workflows
